## ✨ What's New

### 🔧 Build System Switched to Nuitka
- Replaced PyInstaller with **Nuitka** for building the application on all platforms.
- Smaller download sizes and lower memory usage at runtime — no functional or UI changes in this release.
- All platforms (Windows, Linux, macOS Apple Silicon, macOS Intel) are now compiled with Nuitka's standalone mode.

## 🆚 Updated Dependencies
- **yt-dlp:** `2026.07.04`
- **ffmpeg:** `8.1.2`
- **deno:** `2.9.3`
- **SpotDL (Windows only):** `4.5.0`
